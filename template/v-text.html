渲染資料於畫面上

檔名：v-text.html

字串 v-text 與 {{}} (Mustache)

1. html渲染：小明在早餐店吃早餐

2. vue的雙大括號：{{ name }} 在 {{ position }} 吃早餐

3. vue的v-text：在吃早餐

 → 用v-text的好處是可以直接編輯標籤做css樣式設定

原始字串 v-html

{{ rawHtml }}

注意事項

單次綁定 v-once

{{ name }}在{{ position }}吃早餐

進階技巧：表達式

1. 樣板字面值：在雙大括號中加入``(雙反引號)，並在雙反引號中加入文字，即可直接渲染該文字。同時也可以透過js引用變數的方式${}，在文字中混用變數

→ {{ `${name}在${position}吃早餐` }}

程式碼：{{ `${name}在${position}吃早餐` }}

2. 在雙大括號中直接使用js的方法

反轉文字 → {{ text.split('').reverse().join('') }}

程式碼：{{ text.split('').reverse().join('') }}

3. 綁定methods：在雙大括號中直接使用methods的方法

→ {{ say('杰倫') }}

程式碼：{{ say('杰倫') }}

4. 直接在雙大括號中寫入js的運算

→ {{ 1+1 }}

程式碼：{{ 1+1 }}

進階技巧：顯示資料狀態

顯示陣列內容：{{products}}
